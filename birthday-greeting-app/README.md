# Birthday Greeting App

这是一个生日祝福应用程序，允许用户登录并查看生日祝福信息。

## 功能

- 用户登录
- 显示生日祝福卡片

## 技术栈

- React
- TypeScript

## 安装

1. 克隆这个仓库
   ```bash
   git clone <repository-url>
   ```
2. 进入项目目录
   ```bash
   cd birthday-greeting-app
   ```
3. 安装依赖
   ```bash
   npm install
   ```

## 运行

```bash
npm start
```
